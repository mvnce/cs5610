Projects

Dockerized Rapid Development Environment

A solution for setup multiple micro-services for PowerSchool Unified Classroom developers. With dockerized development environment, developers can create an identical isolated environment by using default options which prevents configuration issue and saves setup time. Docker volume maps local source code into container. New change triggers new build task to refresh service inside of the container.

Sharding Service

A PowerSchool Hackathon project that separates data into databases separate from various services and add to a list of responsibilities shard management. Shards can be passed out to the various instance of service. Sharding service distributes incoming database query traffic across a group of backend servers efficiently.
